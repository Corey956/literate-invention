# 日常中常用的命令

## nginx

```bash
# 确认nginx配置是否正确
nginx -t
# 重启nginx
nginx -s reload
```

## 权限

```bash
# 创建用户

#设置文件夹权限
sudo chmod -R 770 /www/server/nginx/html/wordpress/
# 设置文件夹用户
sudo chown -R root:root /www/server/nginx/html/wordpress/

# php-fpm用root启动
php-fpm -R
```
